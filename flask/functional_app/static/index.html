Review Predictor


While this model was originally designed to disambiguate three star reviews it can be used on just about anything.

This model is designed to classify a user review for a good or service as either positive or negative. It does this by taking in a written review, cleaning the data, then pulling out important words or features which are weighted more to one classification or the other.

Please type a review into the text box and hit the big blue Predict! button to get your prediction.
